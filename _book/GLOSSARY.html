Graph

Mathematical object used to model the relationships between objects. To analyse networks, we use graph theory. Google, Facebook or Linkedin are among the companies that use graphs. With the data deluge making peta-octets available, companies and science teams are striving today to make sense of big data, and to extract value from it. A significant amount of these data can be modeled as graphs, where nodes are entities and relationships are whatever connect them to each other. Properties may be attached to the nodes and relationships. Graphs are visualized using “node-link diagrams”, where circles represent the nodes, and lines represent the relationships.

Label

A label is a value associated to a node or an edge. It is a concept used to group nodes or edges into sets. For example if our graph contains many people like John Smith or Paul Durand, we may want to give these nodes a ```Person``` label.

A property is a value associated to a node or an edge. For example, if our graph contains a John Smith, we may have a node with the property ```name``` and the value ```John Smith```.
